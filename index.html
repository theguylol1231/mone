Netlify-ready wrapper

GT4 Reimagined

This project converts the GT4-Reimagined GitHub repository into a web-friendly format you can deploy directly on Netlify.

View Source Repo Download Releases

What this includes

  • A clean landing page describing the original project.
  • Direct links to source code and releases.
  • Netlify configuration for zero-build static deployment.

Deploy on Netlify

  1. Push this repository to GitHub.
  2. In Netlify, click Add new site → Import an existing project.
  3. Select this repository and deploy with default settings.
